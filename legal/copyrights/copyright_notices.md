# Copyright Notice Templates

## Basic Copyright Header (Short)

```text
/*
 * Copyright (c) 2025 [Author/Organization]. All rights reserved.
 * Licensed under AGPLv3. See LICENSE for details.
 */
```

## Full Copyright Header (Detailed)

```text
/*
 * Copyright (c) 2025 [Author/Organization]. All rights reserved.
 *
 * This file is part of NexusZero Protocol ("Project").
 *
 * The Project is provided under dual license: AGPLv3 for open-source use, and a
 * separate commercial license available via direct contact at [email].
 */
```

## Attribution Guidelines

- Include author and organization in header
- If code is from third-party sources, include attribution and license details
- Keep a separate AUTHORS.md for long lists of contributors

## Additional Notes

- Avoid putting private or sensitive legal agreements in the public repo.
