## 2. Proper Trademark Usage

### 2.1 First Reference Rule

On first reference in any document, use the full trademark with the appropriate symbol:

✅ **Correct:**

- "NexusZero Protocol™ is a quantum-resistant zero-knowledge proof system."
- "The NexusZero™ SDK provides easy integration."
- "Privacy Morphing™ technology enables dynamic privacy levels."

❌ **Incorrect:**

- "Nexuszero is a quantum-resistant system." (wrong capitalization, no symbol)
- "The Nexus Zero SDK..." (incorrect spacing)
- "NexusZero's Protocol..." (possessive form)

### 2.2 Trademark Symbols

| Symbol | Usage                   | Example                               |
| ------ | ----------------------- | ------------------------------------- |
| ™      | Unregistered trademarks | NexusZero Protocol™                   |
| ®      | Registered trademarks   | NexusZero® (when registered)          |
| SM     | Service marks           | NexusZero Protocol℠ (service context) |

### 2.3 Grammar Rules

**DO:**

- Use trademarks as adjectives: "NexusZero Protocol™ software"
- Follow with generic noun: "NexusZero™ system", "NexusZero™ network"
- Maintain consistent capitalization: "NexusZero" (capital N and Z)

**DON'T:**

- Use as a noun: ❌ "Download the NexusZero"
- Use as a verb: ❌ "NexusZero your transactions"
- Pluralize: ❌ "NexusZeros"
- Use possessive: ❌ "NexusZero's features"
- Modify spelling: ❌ "NEXUSZERO", "nexusZero", "Nexus Zero"

### 2.4 Capitalization

The official capitalization is:

- **NexusZero** (one word, capital N and Z)
- **NexusZero Protocol** (two words)
- **Privacy Morphing** (two words, both capitalized)

## 3. Logo Usage Guidelines

### 3.1 Logo Versions

| Version              | Use Case                     | File Format   |
| -------------------- | ---------------------------- | ------------- |
| Primary (Full Color) | Marketing materials, website | SVG, PNG      |
| Monochrome (Black)   | Documents, print             | SVG, PNG      |
| Monochrome (White)   | Dark backgrounds             | SVG, PNG      |
| Icon Only            | Favicons, app icons          | SVG, PNG, ICO |

### 3.2 Clear Space

Maintain minimum clear space around the logo equal to the height of the "N" in NexusZero.

```
┌─────────────────────────────────────────┐
│                                         │
│     ┌─────────────────────────┐        │
│     │                         │        │
│     │    [NexusZero Logo]     │        │
│     │                         │        │
│     └─────────────────────────┘        │
│                                         │
│     ↕️ Clear space = height of "N"      │
└─────────────────────────────────────────┘
```

### 3.3 Minimum Size

- **Print**: Minimum width of 1 inch (25mm)
- **Digital**: Minimum width of 100 pixels
- **Favicon**: 16x16 pixels (icon version only)

### 3.4 Prohibited Modifications

❌ **Do NOT:**

- Rotate or skew the logo
- Change logo colors
- Add effects (shadows, gradients, 3D)
- Stretch or distort proportions
- Place on busy backgrounds
- Recreate using different fonts
- Remove or modify any element
- Add outlines or borders
